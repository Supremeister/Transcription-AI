# Транскрибатор — Инструкция пользователя

## Что это такое

Локальное приложение для расшифровки аудиозаписей и автоматического AI-анализа разговоров. Работает без интернета (транскрипция) и с минимальным интернетом (AI-анализ через Groq).

---

## Что умеет

### 1. Транскрипция аудио
- Расшифровывает записи разговоров, звонков, встреч, сессий
- Поддерживает форматы: **MP3, WAV, OGG, M4A, AAC** (до 100 МБ)
- Языки: **Русский** и **English**
- Работает локально на вашем компьютере — файлы никуда не отправляются
- Использует GPU (CUDA) если доступен, иначе CPU

### 2. AI-анализ (Полный отчёт)
После транскрипции автоматически создаётся отчёт с разделами:

| Раздел | Что внутри |
|--------|------------|
| **Задачи и договорённости** | Кто, что и когда должен сделать |
| **Цели разговора** | Чего хотели достичь |
| **Ключевые метрики** | Оцифрованные показатели (%, штуки, дни, рубли) |
| **Ключевые выводы и принципы** | Инсайты, концептуальные различия, принципы |
| **Зоны роста участников** | Где мыслят ситуационно, путают категории, используют размытые формулировки |
| **Динамика** | Сравнение с предыдущими сессиями (прогресс/регресс) |
| **Открытые вопросы** | Что осталось нерешённым |

### 3. Профиль пользователя
Персональный контекст, который подтягивается в каждый анализ. Чем точнее профиль — тем точнее зоны роста и выводы.

### 4. История сессий
Приложение запоминает последние 10 анализов. При следующей сессии AI видит прошлые зоны роста и отслеживает динамику.

## Как пользоваться

### Шаг 1 — Запустить приложение
Открыть `Транскрибатор.exe` (или `win-unpacked/Транскрибатор.exe`).

Подождать пока в шапке появятся зелёные индикаторы:
- **● Сервер готов**
- **● AI готов**
- **● GPU (CUDA)** или **● CPU** — показывает что используется для транскрипции

> Первый запуск может занять 30–60 секунд — идёт загрузка модели Whisper.

### Шаг 2 — Настроить профиль (один раз)
Нажать кнопку **👤 Добавить профиль** в шапке.

Написать:
- Кто вы и чем занимаетесь
- С кем работаете (имена, роли)
- Ваши принципы и фреймворки анализа
- Известные слабые точки собеседников

**Пример профиля:**
```
Я Михаил, бизнес-коуч. Работаю с брокерами по недвижимости.
Принципы анализа:
- Три компонента любой работы: бизнес-логика / результат / измерение
- Строго разделяю: результат ≠ метрика ≠ план ≠ приём
- Постановка задачи: X → для чего → потому что → что получится → что решает

Текущий подопечный: Михаил Камалов, брокер, работает над контрактингом.
Его слабые точки: путает метрику с результатом, мыслит ситуационно.
```

Нажать **Сохранить**. Кнопка станет зелёной — профиль активен.

### Шаг 3 — Загрузить аудио
- Выбрать язык (Русский / English)
- Нажать на зону загрузки или перетащить файл
- Нажать **📝 Транскрибировать**

Время обработки зависит от длины записи и вашего железа:
- ~29 минут аудио на GPU ≈ 3–5 минут
- ~29 минут аудио на CPU ≈ 10–20 минут

### Шаг 4 — Получить результат
После транскрипции автоматически запускается AI-анализ.

Доступные действия:
- **📋 Копировать** — скопировать транскрипцию
- **💾 Сохранить** — сохранить транскрипцию + анализ в .txt файл
- **📊 Повторить анализ** — запустить анализ заново (если нужно обновить с новым профилем)
- **Очистить** — начать заново

## Частые вопросы

**⚠️ Ошибка Groq API: Rate limit**
Groq бесплатный тариф ограничен по токенам в минуту. Приложение автоматически повторяет запрос через 3–5 секунд. Если ошибка осталась — подождите минуту и нажмите **📊 Повторить анализ**.

**Сервер не запускается**
Убедитесь что антивирус не блокирует `whisper_service.exe`. Добавьте папку приложения в исключения.

**Транскрипция пустая**
- Проверьте что выбран правильный язык
- Убедитесь что в аудио есть речь (не только музыка/тишина)
- Попробуйте формат MP3 или WAV

**Профиль не влияет на анализ**
Профиль применяется только к анализу **"Полный отчёт"**. Убедитесь что кнопка профиля в шапке зелёная.

## Конфиденциальность

- Аудиофайлы **не покидают ваш компьютер** — транскрипция полностью локальная
- Для AI-анализа текст транскрипции отправляется в **Groq API** (США) — только текст, не аудио
- Профиль и история хранятся **локально в браузере** приложения
